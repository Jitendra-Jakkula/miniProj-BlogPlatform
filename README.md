# Scribo - Blog Application

![Node.js](https://img.shields.io/badge/Node.js-Backend-green)
![Express.js](https://img.shields.io/badge/Express.js-Framework-black)
![MongoDB](https://img.shields.io/badge/MongoDB-Database-green)
![JWT](https://img.shields.io/badge/JWT-Authentication-blue)

Scribo is a full-stack blogging platform that enables users to create, edit, and manage blog posts with rich-text formatting. The application provides secure authentication, interactive engagement features, and a responsive user experience.

## Tech Stack

### Frontend

* EJS
* Bootstrap
* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### Authentication

* JSON Web Token (JWT)

### Editor

* Quill Editor

## Features

* User registration and JWT-based authentication
* Create, edit, and delete blog posts
* Rich-text editor integration using Quill Editor
* Comments and likes functionality
* Tag-based organization
* Pagination support
* Responsive UI with Bootstrap

## Tech Stack Summary

* Backend: Node.js, Express.js
* Database: MongoDB
* Authentication: JWT
* Frontend: EJS, Bootstrap
* Editor: Quill Editor
